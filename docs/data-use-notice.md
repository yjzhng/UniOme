# Data use notice & disclaimer

Please read before downloading or using UniOme's organism data.

## Academic, non-commercial use

UniOme is a research tool. The organism data archives it distributes are **provided for academic,
non-commercial research and educational use only.** By downloading or using them you confirm that
your use is academic/non-commercial, or that you have obtained the necessary licenses from the
data sources below for any other use.

## UniOme redistributes third-party data under the sources' own terms

UniOme does not own the biological data. Each dataset is fetched or derived from an external
database and is **redistributed here under that database's own license and citation terms**, which
are listed in [data-attribution.md](data-attribution.md). Being able to download data from UniOme does
not grant you any rights beyond what the original source grants — **you are responsible for
complying with each source's license.**

Some sources restrict redistribution and/or commercial use. In particular:

- **KEGG** data (the core `KG_*` annotation columns and all pathway maps) is copyright Kanehisa
  Laboratories. KEGG is free for academic *viewing*, but redistribution and non-academic use are
  governed by KEGG's own license terms (the KEGG FTP academic subscription / a commercial license
  from Pathway Solutions). See <https://www.kegg.jp/kegg/legal.html>.
- **EcoCyc/BioCyc, EnteroBase, DeepLocPro (CC BY-NC-SA), and the Foster and HT-CRISPRi datasets**
  also carry academic-only or non-commercial terms.

See the [redistribution watch-list](data-attribution.md#-redistribution-watch-list) for the full set
and the required actions before any non-academic redistribution.

## Please cite the sources

If you use UniOme's data in published work, cite the primary sources you rely on. Citations (with
DOIs) for every dataset are in [data-attribution.md](data-attribution.md).
